    B, since it has the shortest mean access time.

    $H_1= 0.9$ $H_2=0.8$ $C_1=0.5C_2$ $C_2=0.1M$ $M$ = main memory access time

    $T_{avg} = C_1 + (1-H_1)(C_2 + (1-H_2)M)$ $T_{avg} = 0.5C_2 + 0.1(C_2+0.2M)$ $T_{avg}=0.6C_2+0.02M$ $T_{avg}=0.06M+0.02M=0.08M$
